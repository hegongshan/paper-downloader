### Academic Paper Bulk Downloader for Open Access Venues (APBD4OAV)

#### Install dependencies

```shell
$ pip3 install -r requirements.txt
```

#### Usage

```python
$ 
```

* Example

```shell
$ python3 main.py --venue fast --year 2023
```
